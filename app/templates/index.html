Bucketlist API

This is an api, which you provide data to, to store things you want to do before you die.

How to install the API.

Go to [GitHub: Bucketlist2] and clone the Repository Navigate to the installation folder or create a directory if you have not created one yet, like

$ mkdir bucketlist_api

Move to the created directory

$ cd bucketlist_api

Now clone the repository.

git clone https://github.com/chadwalt/bucketlist2
Make sure you have the lastest python installed. Python Org
Download and install Postgressql if you dont have it. Postgressql

Now Setup Virtual Environment

$ pip install virtualenv
$ pip install virtualenvwrapper
$ export WORKHOME=~/Envs
$ source /usr/local/bin/virtualenvwrapper.sh
$ mkvirtualenv bucketlistapi

Install the requirements

$ pip install requirements.txt

Provide the flask application environment variable

$ export FLASK_APP=run.py

Run the flask server

$ flask run
